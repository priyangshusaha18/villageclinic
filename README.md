# 🏥 Village Clinic Management System

## 📌 Overview
The Village Clinic Management System is a console-based application developed using Core Java, MySQL, and JDBC. It helps manage patient records and track their clinic visits efficiently.

---

## 🚀 Features
- Register new patients
- Record patient visits (diagnosis, medicine, follow-up)
- View complete patient visit history
- Track follow-ups due or overdue

---

## 🛠️ Technologies Used
- Java (Core Java)
- MySQL
- JDBC (Java Database Connectivity)

---

## 📂 Project Structure

---

## ⚙️ Setup Instructions
1. Open MySQL Workbench and run `schema.sql`
2. Update database username and password in `DBConnection.java`
3. Add MySQL Connector/J (JDBC driver) to project
4. Compile and run `Main.java`

---

## ▶️ Usage
Run the program and choose options from the menu:
1. Register Patient  
2. Record Visit  
3. View Patient History  
4. Follow-up Due Today  
5. Exit  

---

## 🎯 Objective
To provide a simple system for managing patient data and follow-ups in rural healthcare clinics using Java and MySQL.
